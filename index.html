3N+1(按我看題目)

題意： 若某數n為2的倍數，則n除以2；若為奇數，則將其乘以3再加1。做一次計算cycle length就加1。最後到n=1時結束。給定i,j且0<i,j<10000,請問i與j中的所有數的cycle length最長為多少?

題解： 設一個max，將i與j中每個數字的cycle length都算一遍，每做一次都與max做比較，若其大於max就將max更新，記得若i>j則須將i與j數字交換。 

            #include <bits/stdc++.h>
            using namespace std;
            int main(){
                int i,j;
                while(cin>>i>>j){
                    cout<<i<<" "<<j<<" ";
                    if(i>j)swap(i,j);
                    int maxx=0;
                    while(i<=j){
                        int n =i;
                        int l = 1;
                        while(n!=1){
                            if(n%2==0){
                                n/=2;
                                l++;
                                continue;
                            }
                            n = 3*n+1;
                            l++;
                        }
                        maxx = max(maxx,l);
                        i++;
                    }
                    cout<<maxx<<endl;
                }
            }
Odd Sum(按我看題目)

題意： 給定一個T和兩個數字a與b，T是test cases且0≤a≤b≤100。可以求出[a,b]中所有奇數的總和嗎?

題解： 設一個sum初始值為0，把a到b中所有數字跑一遍，若為奇數(mod2為1)就加到sum中，最後輸出case與sum就好! 

    #include <bits/stdc++.h>
    using namespace std;
    int main(){
        int t,a,b;
        cin>>t;
        for(int i=1;i<=t;i++){
            cin>>a>>b;
            int sum=0;
            for(int j=a;j<=b;j++){
                if(j%2==1)
                sum+=j;
            }
            cout<<"Case "<<i<<": "<<sum<<end;
        }
    }
